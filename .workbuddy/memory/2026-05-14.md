# 2026-05-14

## Boss 系统大升级

完成了雷电游戏 Boss 的酷炫升级：

### bullets.js 新增子弹类型
- **Wave (波浪弹)**: 沿垂直方向正弦摆动飞行
- **Homing (追踪弹)**: 缓慢追踪玩家位置
- **Split (分裂弹)**: 延迟后分裂为多颗子弹
- **Accel (加速弹)**: 越飞越快
- **Beam (激光弹)**: 长条激光线
- **Ring (环形弹)**: 旋转环状飞行
- **Petal (花瓣弹)**: 花瓣形曲线路径

### boss.js 全面重写
- 3个 Boss 重命名：INFERNO GUARDIAN / PHANTOM STRIKER / VOID OVERLORD
- 每个 Boss 6 种攻击模式 + 1种特殊攻击
- 三阶段 Phase 变身：60% HP → Phase 1，30% HP → Phase 2（暴走）
- 视觉效果：光环旋转、能量触手、引擎喷火、冲刺残影、轨道球、脉冲核心
- Phase 变身时触发 Rage Burst（加速弹全屏爆发）

### main.js 改动
- `bullets.update()` 新增传递 playerX, playerY 参数（供追踪弹使用）
