## 🎮 新功能详解

### 1. 新敌人类型（3种）

#### SpeedyEnemy（高速敌机）
- **特点**: 正常速度3，冲刺速度8
- **行为**: 每2秒冲刺一次，持续20帧
- **视觉效果**: 冲刺时橙色闪光
- **出现关卡**: Level 2

#### RotatingEnemy（旋转敌机）
- **特点**: HP=2，旋转射击
- **行为**: 旋转速度0.05弧度/帧，冷却60帧发射子弹
- **视觉效果**: 十字形状，绿色
- **出现关卡**: Level 3

#### SuicideEnemy（自爆敌机）
- **特点**: 追踪玩家，接近后自爆
- **行为**: 150px开始武装（红色闪烁），80px半径爆炸
- **伤害**: AOE伤害
- **视觉效果**: 菱形 + 骷髅图标
- **出现关卡**: Level 4-5

### 2. 武器系统（4种）

#### NORMAL（普通子弹）
- **伤害**: 1
- **冷却**: 6帧
- **特点**: 基础武器，无限弹药

#### LASER（激光）
- **伤害**: 5/秒（持续光束）
- **能量消耗**: 2点/帧
- **能量回复**: 击杀回复10点
- **特点**: 需要能量管理

#### HOMING（追踪弹）
- **伤害**: 2
- **冷却**: 15帧
- **特点**: 自动追踪最近敌人
- **视觉效果**: 尾焰效果

#### EXPLOSIVE（爆炸弹）
- **伤害**: 3（直接） + 2（溅射）
- **冷却**: 20帧
- **特点**: 命中后AOE伤害（半径60px）
- **视觉效果**: 闪烁效果

**武器切换**: 数字键1-4，HUD显示当前武器和能量条

### 3. 道具系统（4种）

#### SHIELD（护盾）
- **效果**: 吸收3次伤害
- **持续时间**: 10秒（600帧）
- **颜色**: 蓝色

#### RAPID_FIRE（快速射击）
- **效果**: 射速x2
- **持续时间**: 8秒（480帧）
- **颜色**: 黄色

#### BOMB（炸弹）
- **效果**: 清屏 + 高伤害
- **特点**: 一次性使用
- **颜色**: 红色

#### HEALTH（生命恢复）
- **效果**: 回复30%护盾
- **颜色**: 绿色

**掉落概率**:
- 普通敌人: 5%
- 精英敌人: 10%
- Boss: 100%

### 4. 成就系统（8个）

| 成就ID | 名称 | 条件 | 奖励 |
|---------|------|------|------|
| FIRST_BLOOD | 初次见血 | 首次击杀敌人 | 分数+100 |
| BOSS_HUNTER | Boss猎人 | 击杀3个Boss | 生命+1 |
| COMBO_20 | 连击大师 | 达成20连击 | 分数+500 |
| SURVIVOR | 生存专家 | 生存5分钟 | 护盾+1 |
| SHARPSHOOTER | 神射手 | 命中率80%（至少100发） | 武器升级 |
| COLLECTOR | 收集达人 | 收集20个道具 | 炸弹+1 |
| WAR_MACHINE | 战争机器 | 每种武器击杀10个敌人 | 分数+2000 |
| LEGENDARY | 传奇飞行员 | 单局得分50000 | 生命+1 |

**成就通知**: 解锁时屏幕上方滑入通知，2秒后自动消失

**数据持久化**: LocalStorage (`raiden_achievements`)

### 5. 挑战模式（2种）

#### Time Attack（限时模式）
- **时间限制**: 60秒
- **目标**: 尽可能多击杀敌人
- **HUD显示**: 倒计时、击杀数、分数
- **结束条件**: 时间用完或玩家死亡

#### Survival（生存模式）
- **时间限制**: 无限
- **目标**: 尽可能多生存
- **波次系统**: 每波敌人增强（难度递增）
- **HUD显示**: 波次、生存时间、难度倍率
- **结束条件**: 玩家死亡

**排行榜**: LocalStorage (`raiden_challenge_{mode}`)，保存前10名

**选择UI**: 主菜单 → "CHALLENGE" 按钮 → 选择模式

### 6. 视觉升级

#### 背景视差滚动
- **层数**: 3层星星 + 2层云层
- **速度差**: 远景慢，近景快
- **特效**: 星云效果

#### 爆炸粒子效果增强
- **粒子类型**: 核心粒子、火花、烟雾
- **物理效果**: 重力、随机大小/颜色/速度
- **性能优化**: ObjectPool（池大小200）

#### 子弹轨迹特效
- **激光**: 光束效果（透明度渐变）
- **追踪弹**: 尾焰效果
- **爆炸弹**: 闪烁效果

#### 屏幕震动
- **触发时机**: 爆炸、Boss攻击
- **强度**: 根据爆炸大小调整
- **衰减**: 平滑衰减

### 7. Boss攻击模式增强

#### 原有攻击模式
- 直线弹
- 散射弹

#### 新增攻击模式
- **Wave（波浪弹）**: 扇形弹幕（8发）
- **Spiral（螺旋弹）**: 旋转发射（持续3秒）
- **Laser（激光瞄准）**: 预判玩家位置发射激光
- **Summon（召唤小怪）**: 召唤2-3个小怪

#### 阶段转换
- **HP 70%**: 进入第二阶段，速度+20%
- **HP 40%**: 进入第三阶段，攻击频率+50%

## 🚀 使用说明

### 启动游戏

**方法1：直接打开**
```bash
# 在浏览器中打开
open index.html
```

**方法2：HTTP服务器**
```bash
cd /Users/lizhen/WorkBuddy/raiden
python3 -m http.server 8080
# 然后在浏览器访问 http://localhost:8080
```

### 游戏操作

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动飞机 |
| Z / 空格 | 射击 |
| 1-4 | 切换武器 |
| P | 暂停/继续 |
| ESC | 返回主菜单 |

### 游戏流程

1. **主菜单**
   - 点击"START GAME" - 开始正常模式
   - 点击"CHALLENGE" - 进入挑战模式选择
   - 点击"ENDLESS" - 开始无尽模式

2. **挑战模式选择**
   - 点击"TIME ATTACK" - 60秒限时挑战
   - 点击"SURVIVAL" - 无尽生存挑战
   - 点击"BACK" - 返回主菜单

3. **游戏中**
   - 击杀敌人掉落道具
   - 拾取道具获得增益
   - 击败Boss进入下一关
   - 解锁成就获得奖励

4. **游戏结束**
   - 显示统计信息
   - 显示解锁的成就
   - 点击"RESTART"重新开始
   - 点击"MENU"返回主菜单

## 🔧 技术实现细节

### 架构设计

**模块化设计**：
- 每个系统独立成类（AchievementSystem、ChallengeMode、WeaponSystem等）
- 通过构造函数参数传递依赖
- 事件驱动通信（CustomEvent）

**性能优化**：
- ObjectPool模式（粒子系统）
- 帧率无关动画（dt * 60）
- 避免每帧创建对象

**数据持久化**：
- LocalStorage存储成就进度
- LocalStorage存储挑战模式排行榜
- 自动保存/加载

### 关键代码模式

**事件驱动架构**：
```javascript
// 成就系统监听游戏事件
window.dispatchEvent(new CustomEvent('achievement-reward', { detail: reward }));

// 游戏系统响应成就奖励
window.addEventListener('achievement-reward', (e) => {
    this._handleAchievementReward(e.detail);
});
```

**帧率无关动画**：
```javascript
// 所有动画使用 dt * 60 确保在任何帧率下速度一致
this.timer += dt * 60;
this.y += speed * dt * 60;
```

**ObjectPool模式**：
```javascript
// 从对象池获取粒子，而不是新建
const particle = this.pool.get();
if (particle) {
    particle.reset(x, y, vx, vy);
}

// 回收粒子到对象池，而不是销毁
this.pool.recycle(particle);
```

### 跨模块集成

**主游戏循环（main.js）**：
```javascript
_update(dt) {
    // 更新所有系统
    this.player.update(dt, enemies);
    this.enemySystem.update(dt, playerX, playerY);
    this.bullets.update(dt);
    this.powerups.update(dt);
    this.particles.update(dt);
    this.collision.update(dt);
    
    // 更新新系统
    this.combo.update(dt);
    this.achievements.update(dt, gameState);
    
    // 更新挑战模式
    if (this.challenge && this.challenge.isActive) {
        this.challenge.update(dt);
    }
}
```
